📦 Node.js 脚本内部流程

步骤1 NewsAPI 拉取 5 类 × 多国新闻 → 约 50 条原始数据
步骤2 按来源权重 + 关键词频次排序 → 每类取 TOP 3
步骤3 DeepSeek API 逐一分析 15 条新闻 → 第一性原理 + 贝叶斯分析
步骤4 渲染为单个 HTML 文件（内嵌 CSS/JS） → 零依赖，浏览器直接打开
步骤5 自动部署到 Vercel → 获取公开链接

🔑 两个 API 的职责分工

NewsAPI

负责：找新闻（标题、摘要、来源、URL）

不消耗 token，纯 HTTP 请求

DeepSeek API

负责：分析新闻（第一性原理 + 贝叶斯）

每天约 6k-10k token

你需要做的： 注册 Vercel（用 GitHub 账号）、填两个环境变量。其余代码和配置我全搞定。
